# Progressi_Excel

Contenuto della repository:

Funzioni Excel (logiche, matematiche, di ricerca…)

Elaborazione e organizzazione di tabelle

La repository è in aggiornamento continuo e verrà arricchita con nuovi progetti man mano che verranno approfonditi altri strumenti di analisi dei dati.
