Progetto: Forecast e Ottimizzazione di un E-commerce Tech

Sei il data analyst di un e-commerce che vende accessori tech online (mouse, tastiere, cuffie, webcam, ecc.).

L’azienda vuole:

- prevedere le vendite future,
- simulare scenari economici,
- trovare obiettivi di profitto,
- ottimizzare il budget marketing,
- analizzare sensibilità di prezzo e conversioni.
